4. BASE DE DATOS (POSTGRESQL + PGADMIN)
---------------------------------------

Se levanta con Docker.

Desde la raíz ejecutar:

docker compose up -d

Levanta:
- PostgreSQL (localhost:5432)
- pgAdmin: http://localhost:5050 (usuario: admin@admin.com / contraseña: admin123)

PRISMA (ORM)
---------------
Antes de iniciar backend:

cd backend
npm install express cors dotenv jsonwebtoken bcrypt
npm install prisma @prisma/client
npm install multer pdf-parse
npm install -D typescript ts-node-dev
npm install -D @types/node @types/express @types/cors
npm install -D @types/jsonwebtoken @types/bcrypt
npm install -D @types/multer
npx prisma init
npx prisma generate
npx prisma migrate dev --name init
npx prisma studio
Abrirá: http://localhost:5555
